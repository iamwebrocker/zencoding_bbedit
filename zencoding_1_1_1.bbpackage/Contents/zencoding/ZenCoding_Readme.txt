Zen Coding for BBEdit
Author:    Kendall Conrad of Angelwatt.com
Home page: http://www.angelwatt.com/coding/zen-coding_bbedit.php
Zen Coding Home: http://code.google.com/p/zen-coding/

### Installation (for BBEdit version < 10)
Place contents of zip file in the Scripts directory.
~/Library/Application Support/BBEdit/Scripts/

### Installation as .bbpackage (for BBEdit version > 10)
Place the zencoding_[version].bbpackage directory inside
~/Library/Application Support/BBEdit/Packages/
To modify the settings, locate the zen_settings.py file inside
~/Library/Application Support/BBEdit/Packages/zencoding_1_1_1.bbpackage/Contents/zencoding/
(if in finder, use "show contents" right-click command to get "inside" the .bbpackage directory)

To assign a keyboard shortcut for the script go to Window > Palettes > Scripts. Locate the Zen Coding script in the palette that comes up. Click the Set Key button and enter the keyboard shortcut you wish to use. It will tell you if the shortcut is already in use.
